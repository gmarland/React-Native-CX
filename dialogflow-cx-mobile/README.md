# dialogflow-cx-mobile

A react native implementation for Dialogflow CX

## Installation

```sh
npm install dialogflow-cx-mobile
```

## Usage


```js
import { multiply } from 'dialogflow-cx-mobile';

// ...

const result = await multiply(3, 7);
```
